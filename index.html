    Для вёрстки и FrontEnd разработки я использую языки HTML5, CSS3 и JavaScript + SvelteJS.
    Кратко о каждом.

    HTML5 - кости сайта, на которых всё держится.

    CSS3 - язык для красивой обёртки страницы, всегда ведь интеренее смотреть на цветной сайт, а не чёрно белый текст.

    JavaScript (JS) - язык для оживления вашей страницы, добавяет так сказать рефлексы странице.

    И на последок, SvelteJS - удобный инструмент для написания интернет-приложений со скоростью света.
